Receiver project for motion detection alerts over a wireless transmitter this is the base station 
that uses it's wifi connection to signal to other locations to be on alert base station is more of a relay not a camera  /
detector itself.

For wifi we'll use https://learn.adafruit.com/adafruit-airlift-bitsy-add-on-esp32-wifi-co-processor

# Configure

Assuming platform.io is installed run

```
make configure
```
